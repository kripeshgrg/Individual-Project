# Kripesh Gurung Portfolio

A modern glass-style personal portfolio website built using HTML, CSS, and JavaScript.

## Features

- Dark and light mode support
- Automatic theme switching based on user local time
- 6:00 AM to 5:59 PM = light mode
- 6:00 PM to 5:59 AM = dark mode
- Manual theme toggle button
- Two live digital clocks
  - Nepal time
  - User local time
- Sticky navigation bar
- Active menu highlighting on scroll
- Responsive layout
- Glassmorphism-inspired UI
- Contact form section
- GitHub link integration
- CV button
- No photo section
